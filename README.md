# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and Bootstrap**.  
This project showcases my skills, projects, and contact information in a clean and professional layout.

---

## Features
- Responsive design
- Modern and attractive UI
- Easy navigation
- Cross-browser compatibility
- Optimized layout and styling

- HTML5
- CSS3
- Visual Studio Code (VS Code)
- Git
- GitHub
- GitHub Pages
